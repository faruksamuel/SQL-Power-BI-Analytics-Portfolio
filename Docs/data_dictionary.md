# Data Dictionary

## Sales
- SalesOrderNumber: Unique order identifier (duplicates allowed)
- OrderDate: Date of transaction
- ProductKey: Product identifier
- Quantity: Units sold
- Sales: Total revenue for the row
- Cost: Total cost for the row

## Products
- ProductKey: Primary key
- Product: Product name
- Category: Product category
- Subcategory: Product subcategory
- Standard_Cost: Reference unit cost
